# Security Policy

## Supported scope

Security reports for this repository should relate to the public Android demo on the current default branch. The production application distributed through Google Play has a separate release lifecycle and should be identified clearly when reporting an issue.

## Reporting a concern

Please do not publish credentials, personal data, private images, exploit details, or other sensitive material in a public GitHub issue.

Use the developer contact information on the [Dog Breed Identifier Google Play listing](https://play.google.com/store/apps/details?id=com.zaeri.sabourinia.dogbreedidentifier) to report a security or privacy concern privately. Include the affected version, Android version, device type, reproduction steps, and impact where appropriate.

General feature requests and non-sensitive defects may be reported through the repository's GitHub issue tracker.
